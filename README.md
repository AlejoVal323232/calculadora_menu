# Calculadora con Menú Interactivo

Este es un programa desarrollado en C++ que permite realizar operaciones matemáticas básicas, capturando datos del usuario y calculando resultados de manera organizada. Incluye un menú interactivo para gestionar la entrada de números y la visualización de la suma.

## ¿Qué hace este programa?

* **Ingreso de datos:** Permite capturar dos números del usuario de forma sencilla.
* **Cálculo automático:** Procesa la suma de los valores ingresados apenas el usuario lo solicita.
* **Menú interactivo:** Ofrece una interfaz simple para gestionar las operaciones y consultar el resultado, asegurando que los datos existan antes de operar.

## Conceptos que demuestro aquí

* **Estructuras de control:** Implementación de ciclos para mantener el programa en ejecución y procesar la información.
* **Menús interactivos:** Uso de ciclos `do-while` y `switch-case` para permitir al usuario navegar por las opciones del programa de forma fluida.
* **Lógica condicional:** Aplicación de bloques `if-else` para validar que el usuario haya ingresado números antes de intentar realizar la suma o mostrar un resultado.
* **Gestión de estados:** Uso de una bandera booleana para controlar el flujo de la información y evitar errores al usuario.

## Cómo ejecutar este programa

1. Asegúrate de tener instalado un compilador de C++ como Dev-C++.
2. Abre el archivo `calculadora_menu.cpp` en tu editor.
3. Compila y ejecuta el código.
